Our auditing process

  • Code review

    The first step is a review of the code and documentation, to understand the business and contract logic.

  • Static and dynamic analysis

    We run high quality detection software on your code to identify security issues.

  • Manual analysis

    After detecting all issues from our automated tools, our engineers will perform a manual analysis, trying to hack or exploit logic flaws in the contract

  • Reporting

    We'll report all issues to you. After you fix them, we'll grant you a certificate of secure contract.
